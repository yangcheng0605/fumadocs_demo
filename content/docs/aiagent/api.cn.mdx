# API 参考

NXLINK API 提供了多个端点，用于管理外呼任务和通话。在使用这些 API 之前，请确保了解 [API 认证流程](/docs/api-auth)。

## 可用的 API 端点

| 端点 | 方法 | 描述 |
|------|------|------|
| [/openapi/aiagent/call/list](/docs/api/list/post) | POST | 获取通话列表 |

## 请求身份验证

所有 API 请求都需要在请求头中包含认证信息。详细的认证流程请参考 [API 认证说明](/docs/api-auth)。

```javascript
// 请求头示例
headers: {
  "accessKey": "您的accessKey",
  "action": "nxai",
  "bizType": "1",
  "ts": "1693235584320",  // 当前时间戳
  "sign": "计算生成的签名"  // 根据认证文档中的方法计算
}
```

## 错误处理

API 可能返回以下错误代码：

| 状态码 | 描述 |
|--------|------|
| 400 | 请求参数错误 |
| 401 | 认证失败 |
| 403 | 权限不足 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
