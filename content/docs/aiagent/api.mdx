# API Reference

NXLINK API provides multiple endpoints for managing outbound tasks and calls. Before using these APIs, please make sure you understand the [API authentication process](/docs/api-auth).

## Available API Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| [/openapi/aiagent/call/list](/docs/api/list/post) | POST | Get call list |

## Request Authentication

All API requests require authentication information in the request headers. For detailed authentication steps, please refer to the [API Authentication Guide](/docs/api-auth).
